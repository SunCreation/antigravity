Overview

Google Antigravity란?

Google이 개발한 agent-first 로컬 IDE

Firebase Studio에서 축적된 노하우를 바탕으로, 더 강력하고 자유로운 로컬 개발 환경을 제공합니다.

ℹ️
다운로드
antigravity.google/download

Firebase Studio에서 Antigravity로의 마이그레이션을 @fbs-to-agy-export 에이전트로 자동 지원합니다.

Hands-on

실습: Agent와 대화하기

목표: Agent Pane이 익숙해지기

Agent Pane

Agent Pane 기본 사용법

  • 오른쪽 패널의 Agent Pane 클릭
  • 채팅 입력창에 질문 입력
  • AI가 코드를 생성하거나 설명
  • 제안된 변경사항 수락 또는 수정

Exercise 1

실습 1: 코드 설명받기

프로젝트의 파일을 하나 열고, Agent Pane에 입력:

💬
프롬프트 예시
"이 파일의 코드를 설명해줘. 주요 기능이 뭐야?"

→ AI가 코드를 분석하고 구조를 설명해 줍니다

Exercise 2

실습 2: 코드 수정하기

Agent Pane에 구체적인 수정 요청을 입력해 보세요:

💬
프롬프트 예시
"버튼의 색상을 빨간색에서 파란색으로 변경해줘"
"이 함수에 에러 처리를 추가해줘"

→ AI가 변경사항을 제안하고, 수락하면 코드가 수정됩니다

Tips

Agent Pane 사용 팁

✅
좋은 프롬프트
"src/components/Button.tsx의 onClick 핸들러에 로딩 상태를 추가해줘"
❌
나쁜 프롬프트
"이거 좀 고쳐줘"
💡
핵심 원칙
구체적일수록 좋습니다. 파일명, 함수명, 원하는 동작을 명확히 전달하세요.

Common Mistakes

자주하는 실수 TOP 3

  • AI를 맹신하기 — 생성된 코드를 반드시 검증하세요
  • 한번에 너무 많이 요청 — 작은 단위로 나누어 요청하세요
  • 컨텍스트 없이 질문 — 관련 파일을 먼저 열어두세요
