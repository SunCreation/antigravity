점진적 지식 공개 (Progressive Disclosure)
Progressive Disclosure 개념을 설명. AI가 모든 지식을 한 번에 쓰는 게 아니라, 현재 작업에 관련된 지식만 꺼내서 쓴다는 것. 토큰 절약 효과: 전체 로드 시 ~45,000 토큰에서 ~8,000 토큰으로 약 5배 절약. "AI가 똑똑하게 필요한 것만 골라서 씁니다. 우리가 다 신경 쓸 필요 없어요."

자동 vs 수동: 언제 어떻게?

MCP — Model Context Protocol
MCP의 개념을 Agent Skills와 비교하며 설명. Skills가 정적 지식 주입이라면, MCP는 실시간 데이터 접근. 교과서 vs 인터넷의 비유. AI가 Firestore, Auth, Storage에 직접 접근할 수 있다는 것이 핵심. "Skills는 '어떻게 하는지 알려줘'라면, MCP는 '직접 확인해줘'입니다."
