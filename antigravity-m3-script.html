실전 패턴
Pattern 1: Explain first, then...
AI에게 먼저 코드를 설명하게 하고 그 다음에 작업 요청. AI가 코드를 이해한 상태에서 수정하면 부작용 감소. "이해하고 고치는 거랑, 그냥 고치는 거랑 결과가 완전 달라요."

실전 패턴
Pattern 2: Create a plan, don't code yet
Planning Mode 핵심 프롬프트. 'don't code yet'이 마법의 문장. AI를 구조적 사고 모드로 전환. "이 패턴은 정말 어디서든 쓸 수 있어요. 제일 많이 쓰는 패턴."

실전 패턴
