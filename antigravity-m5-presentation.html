Concept

What is Model Context Protocol?

AI 모델과 외부 도구·데이터 소스 간의 표준화된 통신 프로토콜

Anthropic이 제안한 오픈 표준으로, 마치 USB-C가 기기 간 연결을 표준화했듯, MCP는 AI와 도구의 연결을 표준화합니다.

💡
핵심 아이디어
하나의 프로토콜로 모든 AI 도구가 모든 외부 서비스에 연결됩니다.

AI 에이전트가 도구 호출

Client가 도구를 호출하면, Server가 실제 작업을 수행하고 결과를 반환합니다.

Security

MCP 보안 모델

  • 사용자 승인 — 모든 도구 호출 전 사용자 확인
  • 최소 권한 — 필요한 권한만 요청
  • 투명성 — 어떤 도구가 어떤 데이터에 접근하는지 표시
  • 로컬 실행 — MCP 서버는 로컬에서 실행
⚠️
중요
AI가 임의로 외부 서비스에 접근하지 않습니다. 모든 호출은 여러분의 승인을 받습니다.

Prompts

MCP 프롬프트

Firebase MCP Server가 제공하는 특별한 프롬프트를 사용할 수 있습니다.

🚀
/firebase:init
Firebase 프로젝트 초기화를 자동으로 수행합니다. Firestore, Auth, Storage 설정을 한번에.
🚢
/firebase:deploy
Firebase에 프로젝트를 배포합니다. Hosting, Functions, Firestore Rules 등을 배포.

이 프롬프트는 MCP 서버가 자동으로 등록하므로, Agent Pane에서 바로 사용할 수 있습니다.

Concept

Agent Skills란?

AI 에이전트를 위한 지식 모듈

Skills는 에이전트가 특정 기술이나 서비스에 대해 전문가 수준의 지식을 갖게 합니다. 프로젝트 컨텍스트, 모범 사례, 코딩 규칙 등이 포함됩니다.

🎯
Progressive Disclosure
Skills는 토큰 효율성을 위해 progressive disclosure 방식을 사용합니다. 필요할 때만 관련 지식을 로드합니다.

Install

Skills 설치 및 업데이트

간단한 CLI 명령어로 Skills를 설치합니다.

# Skills 설치 npx skills add firebase/agent-skills # Skills 업데이트 npx skills update --all
📥
설치
npx skills add firebase/agent-skills
🔄
업데이트
npx skills update --all

Mechanism

Progressive Disclosure

토큰 효율성을 위한 점진적 지식 공개

1️⃣
기본 개념
간단한 질문 → 핵심만
→
2️⃣
상세 가이드
구현 요청 → 코드 예시
→
3️⃣
심화 정보
복잡한 문제 → 전체 문서
💡
왜 중요한가?
한번에 모든 지식을 로드하면 컨텍스트 윈도우가 낭비됩니다. 필요할 때만 필요한 만큼 로드합니다.
