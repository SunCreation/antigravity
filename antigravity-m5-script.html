What is Model Context Protocol?
MCP의 기본 개념을 소개. Anthropic이 만든 오픈 표준. USB-C 비유가 이해하기 쉬움. "USB-C가 나오기 전에는 충전기마다 다른 케이블이 필요했죠? MCP는 AI 도구 세계의 USB-C입니다."

MCP 이해하기
MCP 아키텍처
클라이언트-서버 아키텍처를 설명. Antigravity가 Client, Firebase가 Server. JSON-RPC 기반 통신. "AI가 도구를 호출하면, 서버가 실제 작업을 하고 결과를 돌려줍니다."

MCP 이해하기
왜 MCP가 중요한가?
MCP가 중요한 3가지 이유: 표준화, 재사용성, 보안. 특히 한 번 만든 MCP 서버를 Claude Code, Cursor, Cline, VS Code Copilot, Windsurf, Gemini CLI 등 여러 도구에서 재사용 가능. "한 번 만들면 어디든 씁니다."
data-section: MCP 이해하기 | 테마: light
8
MCP 이해하기
MCP 생태계
Firebase뿐 아니라 GitHub, Slack, PostgreSQL, Google Drive 등 다양한 MCP 서버가 이미 존재. 커뮤니티가 매일 새로운 서버를 만들고 있다는 점 언급. "생각할 수 있는 거의 모든 서비스에 MCP 서버가 있어요."
data-section: MCP 이해하기 | 테마: dark
9
MCP 이해하기
MCP vs 플러그인
MCP와 기존 플러그인의 차이를 비교. 플러그인은 특정 도구에 종속되지만 MCP는 표준 프로토콜이라 어디든 쓸 수 있음. "VS Code 확장은 VS Code에서만 되죠? MCP는 어디든 됩니다."
data-section: MCP 이해하기 | 테마: white

MCP 이해하기
MCP 보안 모델
MCP의 보안 모델을 설명. 모든 도구 호출은 사용자의 명시적 승인이 필요. AI가 임의로 외부 서비스에 접근할 수 없음. "AI가 멋대로 데이터 삭제하면 안 되니까, 모든 실행 전에 물어봅니다."

Firebase MCP 설정
Firebase MCP Server 설정
설정 전체 흐름을 보여주기. Agent Pane → more_horiz(⋮) → MCP Servers → Firebase → Install 순서. 실제로 화면을 공유하며 어디를 클릭하는지 보여주면 좋음. "따라오세요, 지금 바로 해볼게요."

Firebase MCP 설정
mcp_config.json 설정
실제 mcp_config.json 설정을 보여주기. npx로 firebase-tools@latest mcp를 실행하는 구조. Antigravity에서 Install을 클릭하면 자동 생성됨. "이 설정 파일이 있어야 MCP가 작동합니다. 다행히 자동 생성돼요."
data-section: Firebase MCP 설정 | 테마: dark

Firebase MCP 설정
MCP 프롬프트
Firebase MCP이 제공하는 특별한 프롬프트 소개. /firebase:init으로 프로젝트 초기화, /firebase:deploy로 배포. MCP 서버가 자동으로 등록하므로 Agent Pane에서 바로 사용 가능. "이 프롬프트는 MCP가 알아서 만들어줍니다."

Agent Skills 심화
Agent Skills란?
Skills의 개념을 소개. AI 에이전트를 위한 지식 모듈. 프로젝트 컨텍스트, 모범 사례, 코딩 규칙 포함. progressive disclosure로 토큰 효율성 확보. "Skills를 설치하면 AI가 갑자기 전문가가 됩니다."

Agent Skills 심화
Skills 설치 및 업데이트
npx skills add firebase/agent-skills로 설치. npx skills update --all로 업데이트. 설치 후 Agent Pane에서 자동 인식. "한 줄이면 됩니다. 정말 한 줄이에요."

Agent Skills 심화
Skills vs MCP: 언제 무엇을?
Skills는 정적 지식, MCP는 동적 도구. 둘은 보완 관계. "Skills는 '어떻게 해야 하는지'를 알려주고, MCP는 '직접 해드리겠습니다'라고 합니다. 둘 다 쓰면 최고입니다."
data-section: Agent Skills 심화 | 테마: light
22
Agent Skills 심화
Skills + MCP 시너지
Skills와 MCP를 함께 쓸 때의 시너지를 설명. Skills가 모범 사례를 알려주면 MCP가 바로 적용. 이 조합이 가장 강력한 워크플로우. "이게 Antigravity 확장의 궁극적인 형태입니다."
